## Descrição
Projeto de estudos com Azure AI Foundry para criação e testes de agentes de Inteligência Artificial.

## Objetivos
- Explorar IA Generativa.
- Criar e configurar agentes inteligentes.
- Realizar testes de prompts.
- Avaliar aplicações em suporte técnico e automação.

## Tecnologias
- Microsoft Azure
- Azure AI Foundry
- Inteligência Artificial Generativa
- Prompt Engineering

## Aprendizados
- Criação de agentes de IA.
- Configuração de modelos de linguagem.
- Desenvolvimento de prompts.
- Aplicação de IA em cenários corporativos.
