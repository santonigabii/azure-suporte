# Assistente de Suporte Técnico com Azure AI Foundry

## Descrição

Projeto desenvolvido no Azure AI Foundry para criação de um agente de Inteligência Artificial voltado ao suporte técnico corporativo.

## Função do Agente

O agente atua como Analista de Suporte Técnico, auxiliando usuários com dúvidas e procedimentos relacionados a:

* Microsoft 365
* Outlook
* Microsoft Teams
* Active Directory
* Windows
* SAP
* Salesforce

## Instruções do Agente

Você é um analista de suporte técnico especializado em Microsoft 365, Outlook, Teams, Active Directory, Windows, SAP e Salesforce.

Responda de forma clara, objetiva e em passos curtos.

## Objetivos

* Padronizar orientações técnicas.
* Agilizar atendimentos de suporte.
* Fornecer instruções simples e objetivas.
* Auxiliar na resolução de problemas comuns de TI.

## Tecnologias Utilizadas

* Microsoft Azure
* Azure AI Foundry
* Inteligência Artificial Generativa
* Prompt Engineering

## Aprendizados

* Criação de agentes inteligentes.
* Engenharia de prompts.
* Configuração de agentes no Azure AI Foundry.
* Aplicação de IA para suporte técnico corporativo.
